**REKAPIN** adalah aplikasi manajemen keuangan modern dan kolaborasi tim berdesain premium yang dirancang khusus untuk pengalaman mobile web yang mulus. Dibangun dengan kombinasi teknologi terkini dan sentuhan estetika premium, REKAPIN menawarkan pelacakan anggaran yang presisi serta komunikasi tim interaktif secara real-time.

---

## 🌟 Fitur Utama & Keunggulan

### 🛡️ 1. Desain Premium "Liquid Glass"
* **Harmonisasi Estetika**: Menggunakan palet warna kelas atas, bayangan kartu yang mewah, dan efek transparansi kaca buram (*glassmorphic glass*) di seluruh komponen.
* **Tipografi Modern**: Didukung font kelas profesional untuk keterbacaan tingkat tinggi pada layar perangkat seluler.

### 💬 2. Ruang Kita (Premium Team Chat Channels)
* **Viewport Adaptif**: Panel laci penuh (100% full-height) yang beradaptasi 1:1 terhadap keyboard virtual tanpa adanya gangguan transisi atau pergeseran visual.
* **Bebas Bug Keychain**: Menggunakan elemen `textarea` dinamis tersinkronisasi penuh untuk melenyapkan bug *autofill password keychain* bawaan browser secara permanen.
* **Navigasi Kokoh & Aman**: Mode geser penutup dan penutupan luar (*backdrop tap*) sengaja dinonaktifkan untuk mencegah ketidaksengajaan menutup chat. Menutup panel hanya bisa dilakukan secara eksklusif dengan tombol panah kembali di header premium.
* **Header Elegan**: Dilengkapi status dot aktif beranimasi serta daftar avatar anggota tim yang tumpang tindih secara modis.

### 📝 3. Panel Komentar Dinamis (Comments Sheet)
* **Instagram/IG-Style Swipe Gestures**: Memungkinkan penutupan panel secara intuitif menggunakan tarikan gesek jari ke bawah.
* **Layout Presisi**: Tampilan bubble obrolan bersayap dengan timestamp terintegrasi rapi untuk memberikan nuansa perpesanan premium.

## 🛠️ Panduan Menjalankan Lokal

### Prasyarat
* [Node.js](https://nodejs.org/) terpasang di sistem Anda.

### Langkah Instalasi

1. **Unduh Dependensi**:
   ```bash
   npm install
   ```

2. **Pengaturan Kunci API**:
   Buat atau sunting berkas `.env.local` di direktori utama, lalu konfigurasikan Gemini API Key Anda:
   ```env
   GEMINI_API_KEY=KUNCI_API_GEMINI_ANDA
   ```

3. **Mulai Server Dev**:
   ```bash
   npm run dev
   ```
